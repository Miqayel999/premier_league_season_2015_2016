# ⚽ Premier League SQL Project

## 📌 Project Overview

This project demonstrates my SQL skills by analyzing Premier League football data using PostgreSQL.

The database was designed from scratch and includes information about teams, players, player statistics, and matches. The project contains SQL queries ranging from basic data retrieval to more advanced analytical queries.

---

## 🗄️ Database Schema

The project contains four tables:

- `premier_league_teams`
- `premier_league_players`
- `premier_league_player_stats`
- `premier_league_matches`

Relationships between the tables are implemented using Primary Keys and Foreign Keys.

## 📈 Example Business Questions

This project answers questions such as:

- Which team scored the most goals?
- Which players scored more than their team's average?
- Which team has the oldest squad?
- Who are the Top 10 goal scorers?
- Which defenders scored the most goals?
- Which players never started a match?
- Which team has the most wins, draws, and losses?
